

Step 1: Prepare you IP plan according to the template (if you didn't get the template use the below info)
                    Column 1: Interface
                    Column 2: Site_ID
                    Column 3: IP Address
                    Column 4: Subnet Mask
                    column 5: VLAN
                    column 6: vrf
                    column 7: service_type
Step 2: Open the excel_to_text.html file using your browser (I used JS, and it can be run inside browser without installing additional framework)
Step 3: select the excel file and click on the 'export config file' button
Step 4: all configuration scripts will be exported (downloaded) on the default folder.
